# ⚡ Lightning Pass Backend (REAL DATABASE)

## Setup
```bash
npm install
```

## Seed Database with Sample Venues
```bash
node seedVenues.js
```

## Start Server
```bash
npm start
```

## Test
```bash
curl http://localhost:3000/health
curl http://localhost:3000/api/v1/lightning/venues
```
